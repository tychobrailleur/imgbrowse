# imgbrowse

A Clojure library designed to display images.

## Usage

```
lein run <path/to/pictures>
```

# Keyboard commands 

* `n`, `space`, `→`: Next random picture
* `g`: Open Gimp for the current image
* `f`: Fullscreen
* `p`: Copy path to clipboard.
